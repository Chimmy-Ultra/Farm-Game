## 基礎定位

- **專案類型**：寫實 2.5D 等角視角農場經營遊戲 + 農產品期貨避險機制
- **視角**：固定等角視角（30° 俯視正交相機），**簡化實作難度**
- **美術管道**：AI 生成圖為主，精靈類素材白底/透明背景（詳見 `ART_DIRECTION.md`）
- **用戶角色**：藝術總監 + 產品負責人 + 測試員（**不寫不看代碼**，全部由 AI 實作）
- **時程**：3–4 個月，作品集級別展示
- **技術棧**：TypeScript + Vite + React + R3F + Three.js + Zustand + Tailwind + Zod（+ Howler.js 音效延後）+ localStorage 存檔
- **SSOT**：`src/data/commodities.ts`（由另一邊建立，29 商品定義）
- **時間壓縮**：1 現實日 = 1 遊戲週（一年 ≈ 52 分鐘）— 與 Notion 一致
- **資訊揭露**：半透明（新聞 + 歷史對照，無上帝視角數值） — 採 Notion
- **農場範圍**：單一農場 + 期貨代理參與全球市場 — 採 Notion

## A. 角色與世界

| 項目 | 決策 |
|---|---|

| **體力系統** | 有（每日體力上限，勞動耗體，睡覺回復） |

## B. 內容深度

| 項目 | 決策 |
|---|---|
| **商品數量** | 完整 29 種（以 `docs/` 為準，由另一邊 SSOT） |
| **工具升級線** | 木 → 銅 → 鐵 → 金 四階 |
| **內容深度目標** | Stardew Valley 級別 |
| **期貨系統定位** | **折衷方案**：機制保留 Notion 版（合約月/保證金/平倉/P&L），**UI 分兩個入口**：<br/>- **避險模式**（主推，tutorial 教學重點）：「鎖定我未種的 X 袋玉米未來賣價」<br/>- **投機模式**（次要，需要解鎖）：「不種也能交易，自負風險」 |
| **夜晚機制** | 1) 螢火蟲氛圍粒子；2) 夜間作物（月光下才成長）；3) 老鼠害蟲（未關穀倉會偷糧） |
| **病蟲害** | **獨立 shock 系統**（不併入天氣），與歷史彩蛋連動（2018 中國 ASF、2004 大豆銹病等） |

### 關於「夜間作物」的初步構想（待後續細化）
- 只在夜間推進生長，白天暫停
- 舉例：月見草、夜香花、某些菇類
- 種植選項在作物百科裡特別標註「夜行性」

## C. NPC / 建築 / 教學

| 項目 | 決策 |
|---|---|
| **NPC** | 僅功能性（像按鈕），無劇情/對話/好感度 |
| **側活動** | 完全不做（無釣魚、採礦、節慶、戰鬥等） |
| **建築系統** | 基礎（穀倉、倉庫、圍欄），無裝潢/深度建造 |
| **新手引導** | 內建百科「**農會資料庫**」— 隨時點開查作物/工具/天氣/期貨規則 |

---

## D. 經濟進度

| 項目 | 決策 |
|---|---|
| **起始金幣** | 200 金（白手起家，前期緊繃） |
| **作物定價哲學** | **寫實比例** — 玉米低利潤 → 咖啡/可可等期貨商品高利潤，引導玩家擴張 |
| **解鎖節奏** | **成就制** — 完成特定任務解鎖新作物/工具/建築 |
| **成就獎勵** | **有獎勵**，分層設計由 AI 發揮：<br/>- 初期成就 → 金幣<br/>- 中期成就 → 解鎖新物品（種子/工具/建築） <br/>- 後期成就 → 裝飾徽章/農舍擺設（純榮譽） |
| **投入成本** | **每項詳細計價**（種子/肥料/飼料各自買賣管理） — 真實感強 |
| **勝利條件** | **純沙盒** — 無明確通關，成就系統當軟目標/方向感 |
| **破產處理** | **破產保護**：金幣歸零時重啟但**成就解鎖永久保留** — 不讓挫折趕走玩家 |
